Reflection

Digital Publishing

Learning how to create HTML’s and publishing a website was definitely out of my comfort zone. I have never been great with technology, or really known anything about the coding world. While there were definitely times of frustration, there were also moments where all the puzzle pieces finally came together which felt gratifying to say the least. The hardest part for me was trying to find a platform to create the html, and then I was having issues saving it as the proper file. Once I got to the GitHub portion, at first my website was only showing the “ReadMe” section, and not the html. I found that if I uploaded the html first, and then added the ReadMe after, then it worked. It is all about trial and error, which tested my patience at times. I can definitely see where creating a website from scratch offers more room for creativity, personalization, and freedom rather than using platforms that have pre-made templates. However, I do feel like some elements felt archaic and I am surprised that they haven’t made things easier since the digital world has progressed so much. Overall I am glad that I now know the basics about this side of the digital publishing world, so that in the future if I ever want to create something more intricate it will feel less overwhelming
